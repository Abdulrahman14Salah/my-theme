# a-salah WordPress Theme

A professional, modern, and high-performance WordPress theme tailored for personal portfolios and business websites.

## 🚀 Features

- **Modern Design**: Clean, minimalist aesthetic with dark mode support.
- **Custom Post Types**: Portfolio, Services, and Testimonials built-in.
- **SEO Optimized**: Integrated Schema.org markup, Open Graph tags, and Twitter Cards.
- **Accessibility Ready**: ARIA labels, skip links, and keyboard navigation support.
- **Gutenberg Ready**: Custom block styles and patterns for easy content creation.
- **Performance**: Optimized assets, lazy loading, and minimal dependencies.
- **Customizer Support**: Full control over colors, typography, and layout.

## 🛠 Installation

1. Download the theme zip file.
2. Go to **Appearance > Themes > Add New > Upload Theme**.
3. Upload the zip file and click **Install Now**.
4. Click **Activate**.

## ⚙️ Configuration

### Customizer Settings
Go to **Appearance > Customize** to configure:
- **Site Identity**: Logo and site title.
- **Color Scheme**: Primary, secondary, and background colors.
- **Typography**: Heading and body fonts.
- **Layout Options**: Container width and sidebar position.
- **Social Media Links**: Add your social profiles (Facebook, Twitter, Discord, etc.).

### Widget Areas
The theme supports multiple widget areas:
- **Sidebar**: Standard blog sidebar.
- **Footer Columns 1-3**: Three-column footer layout.
- **Header Widget**: Special area in the header.

### Menus
- **Primary Menu**: Main navigation in the header.

## 🧩 Block Patterns

The theme includes custom block patterns to speed up page building:
- **Hero Section**: A full-width hero area with title, text, buttons, and image.
- **Services Grid**: A 3-column grid to showcase services.

## 💻 Development

### Prerequisites
- WordPress 5.9+
- PHP 7.4+
- Node.js & NPM (for development)

### Commands
```bash
# Install dependencies
npm install

# Watch for changes
npm run dev

# Build for production
npm run build

# Lint CSS/JS
npm run lint
```
